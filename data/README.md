# 数据目录

该目录用于存储项目运行过程中生成的临时数据，包括：

- 爬取的网页内容
- 日志文件
- 中间处理结果

该目录下的内容不会被加入到Git版本控制中，你可以在`.gitignore`文件中看到相关配置。

## 目录结构

系统会自动在此目录下创建以下文件：

- `stock_agent.log` - 日志文件
- `crawl_results_[时间戳].json` - 爬取的网页内容
- 其他临时处理数据

## 清理数据

如果需要清理数据目录，可以直接删除内容，但请保留目录本身：

```bash
# 清空数据目录但保留目录本身
rm -rf data/* 
```
